#  Projeto de Testes Automatizados (JavaScript/Node.js)

Este projeto foi desenvolvido como parte da **Atividade Avaliativa 2 - Projeto de Teste Automatizado** da Unidade Curricular de Qualidade de Software (ADS/SPI) do Centro Universitário Senac-RS.

O objetivo é demonstrar a automação de testes em dois níveis: **Unitário** (para código de backend) e **Funcional em Nível de Sistema** (para interface web).

---

##  Tecnologias Utilizadas

| Categoria | Ferramenta | Descrição |
| :--- | :--- | :--- |
| **Linguagem** | JavaScript (Node.js) | Linguagem principal utilizada nos scripts. |
| **Teste de Sistema** | Playwright | Framework para automação de testes End-to-End (E2E) em navegadores. |
| **Teste Unitário** | Jest | Framework de testes unitários para JavaScript. |
| **Aplicação Testada** | Sauce Demo (`https://www.saucedemo.com/`) | Site demo utilizado para os testes de sistema. |
